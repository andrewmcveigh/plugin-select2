# plugin-select2

Select2 3.4.1 packaged as a Servlet 3.0 plugin. With a couple of clojure/hiccup
helpers.

## Usage

Dependencies:

```clojure
[com.andrewmcveigh/plugin-select2 "0.1.0"]
```

```clojure
> (require '[plugin-select2 :as select2])

> select2/css
> select2/js
> select2/js-debug ; non-minimized version
```
